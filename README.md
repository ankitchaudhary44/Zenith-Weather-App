ZENITH Weather App: Geospatial Intelligence Dashboard

A high-fidelity, futuristic weather telemetry dashboard built with **Next.js 16**, **Framer Motion**, and **Leaflet**. Designed for real-time atmospheric tracking with a "Hyper-Glow" tactical UI.



 Engineering Highlights
- **PWA Integration**: Fully installable web app with Service Worker caching for offline resiliency.
- **Geospatial Telemetry**: Real-time map rendering using Leaflet with custom "Tactical Dark" filters.
- **Responsive Telemetry**: Adaptive charts using Recharts with automatic container resizing.

 Tech Stack
- **Framework**: Next.js (App Router)
- **Styling**: Tailwind CSS + Glassmorphism
- **Animations**: Framer Motion (Radial Glows & Transitions)
- **Maps**: React-Leaflet (Geospatial Sync)
- **Charts**: Recharts (Holographic Telemetry)
